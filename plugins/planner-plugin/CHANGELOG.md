# Changelog

## 1.0.0 - 2026-02-18

### Added

- Initial release of planner plugin
- `create-plan` skill to generate comprehensive implementation plans for Linear issues
- Codebase research phase to identify affected files, modules, and patterns
- Clarifying questions flow to reduce ambiguity before writing the plan
- Structured plan format covering summary, affected areas, approach, data model changes, API changes, testing strategy, risks, and complexity estimate
- Automatic posting of the approved plan as a comment on the Linear issue
