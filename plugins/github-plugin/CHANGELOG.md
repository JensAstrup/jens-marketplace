# Changelog

## 1.0.0 - 2026-02-16

### Added

- Initial release of github plugin
- `address-comments` skill to resolve PR review comments by implementing requested changes
- Automatic retrieval and summarization of PR review comments
- Interactive approval flow for selecting which comments to address
- Parallel subagent-based implementation grouped by file
- Automatic comment resolution after changes are implemented
- Integration with write-tests plugin for optional test generation
